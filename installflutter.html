HOW TO INSTALL FLUTTER
1) make a directory called Android
a) unzip this folder and rename it openjdk
https://github.com/AdoptOpenJDK/openjdk8-binaries/releases/download/jdk8u202-b08/O
penJDK8U-jdk_x64_windows_hotspot_8u202b08.zip
b) download this ok?
https://developer.android.com/studio/#command-tools and then put it like
Android\cmdline-tools\tools

2) download flutter sdk or clone it in src\ folder from
https://flutter.io/docs/get-started/install/windows

3) add the following environment variables
a) JAVA_HOME “C:\Android\openjdk”
b) ANDROID_HOME “C:\Android”
c) ANDROID_SDK_ROOT “C:\Android\cmdline-tools”

4) to %path% add
a) “C:\Android\cmdline-tools”
b) "C:\Android\cmdline-tools\tools\bin"
c) "C:\src\flutter\bin"


5) copy the contents of the:
...cmdline-tools\tools\Lib\_ to cmdline-tools\tools\Lib\

6) Check for sdkmanager --list and then download the correct versions of the
following:
a) sdkmanager “system-images;android-27;default;x86_64”
b) sdkmanager “platform-tools”
c) sdkmanager “build-tools;27.0.3”
d) sdkmanager “platforms;android-27”
e) sdkmanager emulator (optional)

7) DOWNLOAD VS-code and set it up for flutter and dart

8) a path to code.exe as a shortcut or the default installation path should be present in
C:\Users\<user>\AppData\Local\Programs\Microsoft VS Code

9) Run "flutter doctor" in windows cmd



SETTING UP A DEVICE
10) To set up a wireless debugging device

a) add the working directory of adb.exe to your %path% present in C:\ANDROID\platform-tools
(this will be available after you install platform tools with the sdkmanager)
b) adb devices
c) adb tcpip 1234 (you can give any 4 digit port)
d) adb connect [ip address to which mobile is connected]:{port}
e) adb devices (to confirm that the device is connected wirelessly)
